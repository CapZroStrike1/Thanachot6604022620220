ประสบการณ์

2022 - 2025 โปรเจคต่างๆ
  • Project Website ร้านขายหนังสือเขียนโดยใช้HTML, CSS(Framework:Tailwind), Javascript/Typescript (Library: React), Node.js(Framework: Nest.js), MySQL
  • Project ระบบร้านขายอุปกรณ์,อาหาร,ของเล่นสัตว์เขียนโดยใช้ภาษา C# และSQL Server Management Studio(SSMS)
  • Project Game Tank ใช้ภาษา C#
  • ทำ Mail Server โดยใช้โปรแกรม hMailServer และ Mozilla Thunderbird
  • ออกแบบระบบบริการสุขภาพอัจฉริยะสู่บริการอันเป็นเลิศ โดยใช้โปรแกรม Visio สร้าง Diagram และ ออกแบบ UX/UI โดยใช้ Figma

ทักษะความสามารถ

  • เขียนโปรแกรมภาษา Python, C#, C, Javascript/Typescript(Library: React), MySQL, MongoDBได้เบื้องต้น
  • เขียนภาษา HTML, CSS(Framework: Tailwind) ได้เบื้องต้น
  • ใช้ชุดเครื่องมือ Node.js(Framework: Nest.js) ได้เบื้องต้น
  • ใช้แพลตฟอร์มการพัฒนาและทดสอบ API: Postman ได้เบื้องต้น
  • มีทักษะการสื่อสารภาษาอังกฤษ ฟัง พูด อ่าน เขียน ได้เบื้องต้น
